### Update data in a table

```sql
update student set dept='Computer Application' where rlno=1005;
/* changes the department of the student with roll number 1005 to Computer Application */

update course set credits=4 where title like '%Systems%';
/* changes the credits of the course with title containing 'Systems' to 4 */
```

Notes :-
1. LIKE operator is used for pattern matching.
2. % is a wildcard character that matches any sequence of zero or more characters.
3. _ is a wildcard character that matches any single character.
4. LIKE and NOT LIKE are used for pattern matching.

### Delete records from a table

```sql
delete from student where rlno=1005;
/* deletes the record of the student with roll number 1005 */
```

## Alter table commands

- ### Add a new column

```sql
alter table student add email varchar(50);
/* adds a new column email to the student table */
```

- ### Modify column type

```sql
alter table course modify title varchar(50);
/* changes the data type of the title column to varchar(50) */
```

- ### Rename column

```sql
alter table student change dob DateOfBirth date;
/* changes the column name dob to DateOfBirth in the student table */
```

- ### Drop a column 

```sql
alter table student drop column email;
/* drops the email column from the student table */
```
---

## Querying Records (SELECT)

- ### Basic select statement

```sql
select * from student;
/* selects all columns and all records from the student table */
```

- ### Select specific columns

```sql
select rlno,name from student;
/* selects the rlno and name columns from the student table */
```

- ### conditional filtering with WHERE

```sql
select * from student where dept='Computer Science';
/* selects all columns and all records from the student table where the department is Computer Science */
```

- ### Sorting results with ORDER BY

```sql
select * from course order by credits desc;
/* selects all columns and all records from the course table sorted by credits in descending order */
```

- ### Aggregate functions

```sql
select count(*) as no_of_students from student;
/* counts the number of records in the student table */
```

- ### Join Query (View related data)

```sql
select s.name as StudentName, c.title as CourseName, E.grade 
from student s
join enrollment e on s.rlno=e.rlno
join course c on e.crs_id=c.crs_id;

/*selects the name of the student, the title of the course and the grade of the student 
from the student, enrollment and course tables */
```
