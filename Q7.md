## Grant Privilege

### -- Create new user

```sql
create user 'user1'@'localhost' identified by 'password123'
```
---
### -- grant privileges on student table

```sql
grant select,insert,update on student to 'user1'@'localhost';
```
Note:

1.  `select, insert, update` are the privileges that are granted to the user.
2.  `student` is the table on which the privileges are granted.
3.  `user1@localhost` is the user on which the privileges are granted.
4.  `@'localhost'` means the user can connect from localhost.
5.  `@'%'` is wildcard which means the user can connect from any host.
6.  `grant all privileges on *.*` gives all privileges on all tables.
7.  `user1` can't use `delete` or `drop` on `student` table because we have not granted delete or drop privilege.

---

### Revoke privileges on student table

```sql
revoke select,insert,update on student from 'user1'@'localhost';
```

## Transaction control commands
- ### Update and Rollback

```sql
start transaction;
/* This command turns off the autocommit mode temporarily. 
Any INSERT, UPDATE, or DELETE commands you run after this are kept in a temporary sandbox state. 
*/
update employee set salary=27000 where empid=101;

```
```sql
rollback;
-- This command undoes or reverses all the changes you made after start transaction.
```
Note :-
1. The SQL TCL commands are used to control the transactions.
2. To end an active transaction in MySQL, you must use either the COMMIT command or the ROLLBACK command.
--- 

- ### Delete and rollback

```sql
start transaction;

delete from employee where empid = 105;

rollback;
```
---

- ### Commit
```sql
start transaction;

update employee set salary = 25000 where empid = 105;

commit;
```
Note: 

1. `commit` saves the changes permanently.

---

- ### Savepoint
 ```sql
 start transaction;

 savepoint sp1;
 
 delete from employee where empid = 105;

 rollback to sp1;   

 ```
Note: 

1. Savepoint works like checkpoints.
2. If we rollback to a savepoint, all the changes made after that savepoint are undone.
3. We can rollback to a savepoint multiple times.
