# Creation of database schema using DDL(create tables, set constraints, enforce relationships, create indices, delete and modify schema objects).

---

### **To create a database schema and use it**

```sql
create database UniversityDB;
use UniversityDB;

```
---

### **To create tables with constraints**

```sql
create table instructor(
    inst_id int primary key, 
    name varchar(20) not null, 
    dept varchar(20)
    );

create table course(
    crs_id int primary key, 
    title varchar(20) not null, 
    credits int check(credits>0), 
    inst_id int, 
    foreign key (inst_id) references instructor(inst_id) on delete set null on update cascade
    );

create table student (
    rlno int primary key, 
    name varchar(20) not null, 
    dob date,
    dept varchar(20)
    );

create table enrollment (
    rlno int, 
    crs_id int, 
    grade char(2), 
    primary key (rlno, crs_id), 
    foreign key(rlno) references student(rlno) on delete cascade on update cascade, 
    foreign key (crs_id) references course(crs_id) on delete cascade on update cascade
    );

```
Notes :-  
1. On delete and on update action(cascade, set null, restrict, no action, set default).

1. On delete cascade is used to delete the rows in the child table when the rows in the parent table are deleted.

1. On update cascade is used to update the rows in the child table when the rows in the parent table are updated.

1. On delete cascade on update cascade is used to delete and update the rows in the child table when the rows in the parent table are deleted and updated.

---

### **Create indices on the table**

```sql
create index idx_student_dept on student(dept);
show index from student;
```
Notes :-  
1. Create index is used to create an index on the table.    
1. Index is used to speed up the query performance by maintaining an sorted copy of the table. 
1.  It sorts all the values in the column and stores the address of the row in the index. 

---

### **Modify or delete the table**
