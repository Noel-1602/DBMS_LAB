# Creation of Procedures, Triggers and Functions.

## Creation of a stored procedure

### Insert a new student record
 ```sql
delimiter //
create procedure AddStudent(
in prlno int,
in pname varchar(20),
in pdept varchar(20),
in pdob date
)
begin
    insert into student (rlno,name,dept,DateOfBirth) values (prlno,pname,pdept,pdob);
    call DisplayStudent();
end //
```
```sql
delimiter ;
call AddStudent(120,'Anu Nair','CSE','2004-07-15');
```
Note: 
1. A stored procedure is a prepared SQL code that can be stored and reused.
2. "call" keyword is used to execute a stored procedure.
3. You use the DELIMITER command in SQL to change the standard statement terminator (the semicolon ;) to a different character or string.
4. Can accept input parameters and return result set or values.
5. Supports transaction control.

### Procedure to display students of a department
```sql
delimiter //
create procedure ShowStudByDept(
in deptname varchar(50)
)
begin
    select rlno,name,dept
    from student
    where dept = deptname; 
end //
```
```sql
delimiter ;
call ShowStudByDept('CSE');
```

### Procedure to calculate the sum of two numbers
```sql
delimiter //
create procedure AddNumbers(
in num1 int,
in num2 int,
out sum int
)
begin
    set sum = num1 + num2;
end //
```
```sql
delimiter ;
call AddNumbers(10,20, @sum);
select @sum;
```
```
Output:
| @sum |
|------|
| 30 |  
```
Note:
1. IN : Used to pass value to the procedure
2. OUT : Used to return value from the procedure
3. INOUT : Used to pass and return value from the procedure
4. @sum is a user defined variable and its value is assigned in the procedure

## Creation of Trigger

### Before insert Trigger - Check Age limit

```sql
delimiter //
create trigger checkage
before insert on people
for each row
begin
    if new.age < 0 then
    set new.age = 0;
    end if;
end //
delimiter ;
```
```sql
insert into people(pid,name,age) values (1,'abc',-10);
```
```
Output:
| pid | name | age |
|-----|------|-----|
| 1 | abc | 0 |
```
Note:
1. Triggers are blocks of code that automatically execute in response to certain events.
2. BEFORE and AFTER are timing specifiers.
3. NEW and OLD are special keywords that refer to the new and old values of the row being affected.

### Before insert - Check Credit Limit

```sql
delimiter //
create trigger checkcredit
before insert on course
for each row
begin
    if new.credit > 6 then
    signal sqlstate '45000' 
    set message_text = 'Credit limit exceeded';
    end if;
end //
delimiter ;
```
```sql
insert into course
values (110,'DBMS',7,100);
```
```
Error Code: 1644. Credit limit exceeded

```
Note: 
1. SIGNAL SQLSTATE can be used to raise an error or warning.
2. message_text is used to display the error message.
3. Error 45000 is user defined error
4. Error code 1644 is a generic error code used for user-defined errors.

## Creation of a function

- ### Sum of two numbers

```sql
delimiter //
create function sum_of_two(num1 int, num2 int)
returns int
deterministic
begin
    return num1 + num2;
end //
delimiter ;
```
```sql
select sum_of_two(10,12) as sum;
```
```
Output:
| sum |
|-----|
| 22 | 
```

Note:
1. A function is a block of code that returns a value
2. DETERMINISTIC means that the function will always return the same result for the same input values
3. NOT DETERMINISTIC means that the function may return different result for the same input values
4. Functions cannot perform DML operations
5. Functions cannot perform DDL operations
6. Functions can accept input parameters and return result set or values.
7. Functions can be used in SELECT statements
8. Functions cannot call stored procedures
9. Must return exactly one value

### Function to calculater total courses by student

```sql
delimiter //
create function total_courses(prlno int)
returns int
deterministic
begin
    declare course_count int;
    select count(*) into course_count from enrollment
    where rlno = prlno;
    return course_count;
end //
delimiter ;
```
```sql
select name, total_courses(rlno) as courses_enrolled
from student;
```
```
Output:
+----------+------------------+
| name     | courses_enrolled |
+----------+------------------+
| Anu Nair |                0 |
| Anjali S |                2 |
| Rahul V  |                1 |
| Divya P  |                1 |
| Arun K   |                1 |
| Neha R   |                0 |
| Reetha   |                0 |
+----------+------------------+
```

Note :
1. Functions can call other functions
2. Declare variables using DECLARE keyword
3. into keyword is used to store the result of a query in a variable
4. return keyword is used to return a value

### Function to calculate Average Credit of all courses taken by a student

```sql
delimiter //
create function Avg_credits()
returns decimal(4,2)
deterministic
begin
declare avg_credits decimal(4,2);
select avg(credits) into avg_credits
from course;
return avg_credits;
end //
delimiter ;
```
```sql
select Avg_credits() as avg_credits;
```
```
output :
+-------------+
| avg_credits |
+-------------+
|        3.75 |
+-------------+
```
