# Practice of SQL commands for creation of views and assertions.

### Create view

- ### Simple view

```sql
create view student_view as
select rlno,name, dept from student;
```
---

- ### Conditional view

```sql
create view cse_stud as
select rlno,dept from student
where dept = 'Computer Science';
```
---

- ### join based view

```sql
create view stud_course as
select s.name as StudentName,c.title as CourseTitle, e.grade
from student s join enrollment e on s.rlno = e.rlno join course c on e.crs_id=c.crs_id;
```
---
## Modifying and dropping views

- ### Update a view

```sql
create or replace view stud_view as 
select rlno,name,dept,DateOfBirth
from student;
```

- ### Drop a view

```sql
drop view cse_stud;
```
---

## Check constraints

```sql
create table student2(rlno int primary key,name varchar(20),age int check(age>=18));
```
