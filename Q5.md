# Implementation of various aggregate functions, Order By, Group By & Having clause in SQL.
---

### Average course credits

```sql
select avg(credits) as Average_credits from course;
```
---

### Minimum and Maximum course credits

```sql
select min(credits) as Minimum_credits,max(credits) as Maximum_credits from course;
```
---

### ORDER BY clause

- ### List all students in alphabetical order

```sql
select * from student order by name;
```
Note :-
order by is ascending order by default.
---

- ### List courses by descending credit values

```sql
elect * from course order by credits desc;
```
---

## GROUP BY clause

- ### No. of students in each department

```sql
select dept,count(*) as no_of_students from student group by dept;
```
Note :-
1. The group by clause is used to group rows that have the same values in one or more columns into a summary row.
2. It is used with aggregate functions to perform operations on groups of rows.
3. The ORDER BY, HAVING, and SELECT clauses are executed in the following order:-
    1. FROM clause
    2. WHERE clause
    3. GROUP BY clause
    4. HAVING clause
    5. SELECT clause
    6. ORDER BY clause

---

- ### Total enrolled students in each course

```sql
select c.title as course_name,count(e.rlno) as total_enrollments from course c join enrollment e on c.crs_id=e.crs_id group by c.title;
```
---

## HAVING clause

- ### List departments with more than 2 students

```sql
select dept,count(*) as no_of_students from student group by dept having count(*) > 2;
```

Note :-
1. The HAVING clause is used to filter groups created by the GROUP BY clause. It is similar to the WHERE clause, but it is used to filter groups instead of rows.

---

- ### List courses with more than 1 student enrolled

```sql
select c.title as course_name,count(e.rlno) as total_enrollments from course c join enrollment e on c.crs_id=e.crs_id group by c.title having count(*) > 1;
```
---

### Aggregate + ORDER BY + GROUP BY + HAVING

```sql
select c.title as course_name,count(e.rlno) as students_enrolled,avg(c.credits) as avg_credits from course c join enrollment e on c.crs_id=e.crs_id group by c.title having count(e.rlno) >=2 order by students_enrolled desc;
```
